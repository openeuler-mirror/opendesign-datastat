# 开源社区贡献面板

## 介绍

vue3 模板库, 使用 vite 构建，并集成 Scss + Vue Router + Pinia + Axios

## 软件架构

Vue 3 + TypeScript + Vite + Scss + Vue Router + Pinia + Axios

## 安装教程

1. 推荐使用[pnpm](https://pnpm.io/installation)安装，当前你也可以使用 npm、yarn
2. 安装依赖 `pnpm install`
3. 运行项目 `pnpm dev`
